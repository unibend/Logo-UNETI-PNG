# Logo UNETI PNG

Este userscript reemplaza automáticamente el logo principal de la página web de la UNETI por una versión en formato PNG. 

## Instalación

1. Asegúrate de tener instalada una extensión de gestión de userscripts en tu navegador, como [Tampermonkey](https://www.tampermonkey.net/) o Greasemonkey.
2. Haz clic en el siguiente enlace para instalar el script:

[➡️ Instalar el script](https://raw.githubusercontent.com/unibend/Logo-UNETI-PNG/main/unetipng.user.js)

Tampermonkey abrirá una nueva pestaña para confirmar la instalación. Simplemente haz clic en el botón **"Instalar"** y recarga la página de la UNETI usando `F5`.
